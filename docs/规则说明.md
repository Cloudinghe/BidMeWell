# 现代精确叫牌法规则说明

本文档详细介绍现代精确叫牌法的规则体系。

## 一、体系概述

### 1.1 历史背景

精确叫牌法由美籍华人魏重庆于1965年发明，是一种以限制性叫牌为核心的桥牌叫牌体系。现代精确（Standard Modern Precision, SMP）是在传统精确基础上的改进版本。

### 1.2 核心特点

| 特点 | 说明 |
|------|------|
| **1♣强开叫** | 16+ HCP，约定性开叫，与梅花张数无关 |
| **限制性开叫** | 除1♣外所有开叫均为11-15 HCP |
| **弱无将** | 1NT = 13-15 HCP 均型 |
| **转移示强** | 1♣后用转移叫显示5张以上套 |
| **二盖一逼局** | 2阶新花应叫逼叫到局 |

### 1.3 点力计算

#### 高牌点 (HCP)

| 牌 | 点力 |
|----|------|
| A | 4点 |
| K | 3点 |
| Q | 2点 |
| J | 1点 |

全副牌共 40 点，平均每人 10 点。

#### 支持点

有将牌配合时计算：

```
支持点 = HCP + 牌型点
- 双张 = +1点
- 单张 = +2点
- 缺门 = +3点
```

#### 控制点

满贯试探时使用：

```
- A = 2控制
- K = 1控制
- 小满贯需联手 8-9 控制
```

## 二、开叫规则

### 2.1 开叫一览表

| 开叫 | 点力 | 张数/牌型 | 说明 |
|------|------|----------|------|
| 1♣ | 16+ | 任意 | 强牌约定叫（排除22-24均型） |
| 1♦ | 11-15 | ♦≥2 | 限制性开叫，无5张高花 |
| 1♥ | 11-15 | ♥≥5 | 5张高花开叫 |
| 1♠ | 11-15 | ♠≥5 | 5张高花开叫 |
| 1NT | 13-15 | 均型 | 弱无将 |
| 2♣ | 11-15 | ♣≥6 | 限制性开叫 |
| 2♦ | 视约定 | 视约定 | 特殊约定叫 |
| 2♥/2♠ | 5-10 | ≥6 | 弱二高花 |
| 2NT | 22-24 | 均型 | 强无将 |

### 2.2 1♣强开叫

**定义**：16+ HCP，约定性开叫，与梅花张数无关。

**例外情况**：
- 22-24 HCP 均型 → 开叫 2NT
- 25-27 HCP 均型 → 开叫 3NT

**后续叫牌**：
1. 1♦ = 0-7 HCP 示弱
2. 1♥ = 转移黑桃，8+ HCP
3. 1♠ = 转移低花，8+ HCP
4. 1NT = 8-10 HCP 均型
5. 2♣ = 11+ HCP 逼局
6. 2♦ = 8-10 HCP, 4441牌型

### 2.3 1♦限制性开叫

**定义**：11-15 HCP，至少2张方块，无5张高花。

**后续叫牌**：
1. 1♥/1♠ = 6+ HCP，4张以上套
2. 1NT = 6-10 HCP，无4张高花
3. 2♣ = 11+ HCP，5张以上梅花，逼局
4. 2♦ = 6-10 支持点，4张以上方块

### 2.4 1♥/1♠高花开叫

**定义**：11-15 HCP，5张以上高花。

**后续叫牌**：
1. 1NT = 6-12 HCP，半逼叫
2. 2♣/2♦ = 11+ HCP，逼局
3. 简单加叫 = 6-9 支持点
4. 限制性加叫 = 10-12 支持点
5. Jacoby 2NT = 13+ HCP（纯HCP）

### 2.5 1NT弱无将

**定义**：13-15 HCP 均型牌。

**后续叫牌**：
1. 2♣ = 斯台曼，询问高花
2. 2♦ = 转移红心
3. 2♥ = 转移黑桃
4. 2♠ = 转移梅花
5. 2NT = 8-10 HCP，邀请
6. 3NT = 11-15 HCP，直接进局

## 三、竞争叫牌

### 3.1 技术性加倍

| 位置 | 点力 | 说明 |
|------|------|------|
| 1阶 | 11+ HCP | 支持未叫花色 |
| 2阶 | 15+ HCP | 支持未叫花色 |
| 平衡 | 可轻2-3点 | 平衡位置 |

### 3.2 争叫

| 类型 | 点力 | 说明 |
|------|------|------|
| 1阶争叫 | 8+ HCP | 5张以上好套 |
| 2阶争叫 | 10+ HCP | 5张以上好套 |
| 1NT争叫 | 15-18 HCP | 均型牌 |

### 3.3 负加倍

对手争叫后加倍，表示未叫高花4-4分布。

## 四、满贯叫牌

### 4.1 Blackwood问A

4NT问A，应答方式：

| 应答 | A的数量 |
|------|---------|
| 5♣ | 0或4个 |
| 5♦ | 1个 |
| 5♥ | 2个 |
| 5♠ | 3个 |

### 4.2 Gerber问A

NT序列中4♣问A，应答方式：

| 应答 | A的数量 |
|------|---------|
| 4♦ | 0或4个 |
| 4♥ | 1个 |
| 4♠ | 2个 |
| 4NT | 3个 |

## 五、牌型定义

### 5.1 均型牌

无单缺的牌型：

- 4-3-3-3
- 4-4-3-2
- 5-3-3-2（5张为低花）

### 5.2 半均型牌

有一个双张的牌型：

- 5-4-2-2
- 6-3-2-2

### 5.3 不均型牌

有单缺的牌型：

- 5-4-3-1
- 6-4-2-1
- 7-3-2-1

### 5.4 好套定义

前五张中有三张大牌（A、K、Q中的三张）。
