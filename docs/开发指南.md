# 开发指南

本文档介绍如何参与现代精确叫牌练习系统的开发。

## 开发环境设置

### 1. 克隆项目

```bash
git clone <repository-url>
cd BidMeWell
```

### 2. 创建虚拟环境

```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
# venv\Scripts\activate   # Windows
```

### 3. 安装开发依赖

```bash
pip install -r requirements.txt
pip install -e ".[dev]"
```

## 核心模块说明

### 1. 点力计算 (precision_formulas.py)

提供桥牌点力计算功能：

```python
from precision_formulas import hcp, controls, is_balanced

# 计算高牌点
hcp_value = hcp(hand)  # 返回 int

# 计算控制点
controls_value = controls(hand)  # A=2, K=1

# 判断均型
if is_balanced(hand):
    print("均型牌")
```

### 2. 发牌引擎 (practice/redeal/)

提供随机发牌功能：

```python
from practice.redeal import Deal

# 创建发牌器
dealer = Deal.prepare({})

# 发一副牌
deal = dealer()

# 访问各方位的牌
hand_north = deal.north
hand_south = deal.south

# 获取牌力信息
print(hand_south.hcp)     # 高牌点
print(hand_south.shape)   # 牌型 (spades, hearts, diamonds, clubs)
```

### 3. XML规则解析 (practice/xml_parsing/)

将XML规则解析为可执行的判断逻辑：

```python
from practice.xml_parsing.xml_parser import XmlReaderForFile

# 加载规则
reader = XmlReaderForFile("rules/modern_precision.xml")
bids = reader.get_bids_from_xml()

# 判断手牌是否符合某叫品
if bids["1c"].accept(hand):
    print("可以开叫1♣")
```

## XML规则格式

### 基本结构

```xml
<bid id="unique-id">
  <value>1c</value>
  <desc>中文描述</desc>
  <and>
    <evaluation>
      <hcp><min>16</min></hcp>
    </evaluation>
    <shape type="clubs"><min>5</min></shape>
  </and>
</bid>
```

### 逻辑运算

```xml
<!-- AND 逻辑 -->
<and>
  <condition1/>
  <condition2/>
</and>

<!-- OR 逻辑 -->
<or>
  <condition1/>
  <condition2/>
</or>

<!-- NOT 逻辑 -->
<not>
  <condition/>
</not>
```

### 点力条件

```xml
<!-- 高牌点范围 -->
<evaluation>
  <hcp><min>11</min><max>15</max></hcp>
</evaluation>

<!-- 支持点范围 -->
<evaluation>
  <points><min>6</min><max>9</max></points>
</evaluation>
```

### 牌型条件

```xml
<!-- 均型牌 -->
<shape type="general">balanced</shape>

<!-- 花色张数 -->
<shape type="hearts"><min>5</min></shape>

<!-- 公式表达式 -->
<shape type="formula">hearts + spades >= 8</shape>
```

## 添加新规则

### 1. 编辑XML文件

编辑 `rules/modern_precision.xml`，添加新的 `<bid>` 元素。

### 2. 验证规则

```bash
python scripts/validate_rules.py
```

### 3. 测试

```bash
python scripts/run_practice.py --auto --boards 5
```

## 常见问题

### Q: 如何调试XML规则？

A: 使用 `validate_rules.py` 脚本验证规则：

```bash
python scripts/validate_rules.py -v
```

### Q: 如何添加新的点力计算方法？

A: 编辑 `src/precision_formulas.py`，添加新函数。

### Q: 如何支持其他叫牌体系？

A: 创建新的XML规则文件，使用 `-r` 参数指定：

```bash
python scripts/run_practice.py -r rules/sayc.xml
```

### Q: 导入模块失败怎么办？

A: 确保在项目根目录下运行，或添加到 PYTHONPATH：

```bash
export PYTHONPATH="${PYTHONPATH}:$(pwd)/src"
```
